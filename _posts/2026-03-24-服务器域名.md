最近因为要把博客网站从nas上搬运到阿里云服务器，又重温了一遍如何完美搬迁wordpress整站。其实搬运wordpress博客无非就是以下两种情况：1. 更换服务器，不换域名 2. 更换域名

下面我分别介绍一下如何完美搬迁wordpress博客

[TOC]



## 1. 更换服务器，不换域名

这种情况下相对比较简单，三步即可

1. 备份原服务器wordpress文件夹下所有内容
2. 进入phpmyadmin备份全部mysql数据库文件
3. 在新站点上传备份文件并恢复备份的数据库文件

简单三步即可完美迁移!

## 2. 更换域名

这种情况也只需要两步即可

1. 进入phpmyadmin后台
2. 执行如下代码

```mysql
UPDATE wp_options SET option_value = replace(option_value, 'www.old.com','www.new.com') ;    
UPDATE wp_posts SET post_content = replace(post_content, 'www.old.com','www.new.com') ;    
UPDATE wp_comments SET comment_content = replace(comment_content, 'www.old.com', 'www.new.com') ;    
UPDATE wp_comments SET comment_author_url = replace(comment_author_url, 'www.old.com', 'www.new.com') ;
```

其中`www.old.com`是你原来的网站域名，`www.new.com`是新的网站域名，只需要执行如下mysql代码即可替换wordpress中所有的域名，包括文章、图片等等整站均可迁移至新的域名，简单明了

Tips:如果同时更换了服务器,只需要按照提前执行一遍第一步即可





很多时候我们会遇到更换域名的需求，比如嫌老域名长度太长、不好记忆，或者是发现更符合心意的网站。怎么给WordPress网站更换域名，前提是在[宝塔面板](https://so.csdn.net/so/search?q=宝塔面板&spm=1001.2101.3001.7020)+WordPress环境搭建的网站。

### 第一步：新[域名解析](https://so.csdn.net/so/search?q=域名解析&spm=1001.2101.3001.7020)到空间服务器

所有工作之前要先把新域名解析到空间服务器，不同服务商，解析方法大同小异，添加2条A记录即可。

### 第二步：宝塔面板中给网站添加新域名

在宝塔面板》网站设置》域名管理中添加新的域名：例如new.com和www.new.com。老域名的网站可以删除掉，但不能删除老域名底下根目录和数据库的文件（当删除老域名的网站时会提示是否删掉根目录和数据库的文件，ftp，都不能勾选删除）。不然后面的旧网站的数据库和内容会完全消失。



### 第三步：网站新域名SSL设置

宝塔面板》网站设置》SSL中，先关闭原来网站的SSL，然后再给新域名申请新的SSL证书。这样我们后面在设置域名的时候，直接填写http**s**：//而不是http：//



### 第四步：宝塔面板中更改数据库【Wp_option表单】

宝塔面板》数据库》管理中，通过phpMyAdmin修改网站数据库【wp_options】表中的2个字段【siteurl】、【home】值为新的域名。前面我们已经给新网站设置部署过SSL，所有这里的网站就是带https的，而不是http。

如果前面网站域名解析成功的话，设置之后可以访问新的网站
